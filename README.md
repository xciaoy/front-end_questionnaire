# 薪資數據圖表調查

## Introduction

使用 Vue 實作薪資數據圖表調查，串接 六角學院 所提供 API 資訊。

## Project 

* npm install
* npm run serve

## Use Technology & Library

* Vue CLI
* ESLint
* SCSS
* bootstrap
* chart.js
* vue-chart-3
* vue-meta
* vue-router
